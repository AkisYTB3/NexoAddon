# Custom Block Experience

Drop Experience from Custom Block!

```yaml
# Example

Mechanics:
  custom_block:
    type: NOTEBLOCK
    custom_variation: 6
    model: test_block
    drop:
      silktouch: false
      experience: 5            # Drop 5 experience
      loots:
      - nexo_item: test_block
        probability: 1.0
```
